Grid based design?

Grid based design basically covers all web design. On your web page you must organize your information into a grid in order for the user to easily understand the information. A basic setup works on 12 colums wide and then is generally broken into 3 or 4 major columns

Responsive web pages?

Well, all the hype is about the fact that a web page can change depending on the siz of the browser the page is being viewed on. In your code you can specify a page to change or be seen in a certain way when at a certain size, say 500px. This makes it easy to build a page for all tablets and screens in one. Its called media specifier
