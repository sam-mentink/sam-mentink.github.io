How does Java Script Compare to HTML and CSS?

So far, it appears that JS is far more complex and able to many more functions than HTML and CSS. I would liken it to an machine that can be setup to do many different things, ti is a matter of learning the dynamics of those things in order to construct what actions to be carried out. Whilst HTML and CSS is more like a frame work that you arrange.

Control Flow and Loops.

Control Flow and loops, using for loops and arrays etc. In one way this is a little like using maps on your phone to get from point A to point B. Once you enter in the destination the app will show you a number of ways to get there - depending on traffic, time etc you will pick a way to get there. If on the way something comes up and you have to stop or take a detour, the map will reorientate the path to suit.

acessing data

Object literals are notations that represent a fixed value where as Arrays are clusters of information that may be on different types - Like numbers, true/false statements, and variables.

Functions?

Functions are ways to carry out actions, we can call a function and define the action for that function to carry out, within this we can include variables.
