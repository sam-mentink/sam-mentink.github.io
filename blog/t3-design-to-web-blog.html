Responsive Website?

A responsive website organises itself to the screen you look at the website on. So if your on a phone which has a smaller screen, the website will fit the screen your looking on and youll be able to navigate easily. If you were on a normal desktop site it would be hard to read the website and probably difficult to find what your looking for.

What is Mobile First Design?

Mobile first design is done in order to reduce data flow and processign power when searching websites. If you start with desktop websites first mobile and tablet devices will end up loading all the same data when it may not be needed for a smaller device. Its easier to build for smaller screens first and using media queries build up.

What are Frame works

Frameworks are pre organized website structures. The frame works define the way in which a website can be build so it makes it easier to build using the pre defined layout.

Wireframes

Wireframes are used, much like architecture plans for a house. Having a well thought out plan of what you want to do makes it easier to achive and saves time. It allows you to be more methodical in your approach to building your site.
